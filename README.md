# turborepo-yarn-example

A turborepo example with yarn v1 workspace.

## Includes

- react-ts-lib (tsup)
- ts-lib (tsup)
- ssr app (next)
- spa app (vite)
- node server (nodemon)

## Tools

- test: vitest
- docs: ladle
- eslint: alloy-eslint
